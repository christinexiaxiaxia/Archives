“Saving Grace”, draft

Interactive website.

This is one of my early drafts of my Saving Grace where I wanted to use the tools of the Web¹ to create a more lively and engaged experience of handling artifacts preserved in an archive. This version aimed to make use of the keyboard and would require a user to hold down multiple keys at once to listen to audio tracks. More about this project can be found in The Archive.²
