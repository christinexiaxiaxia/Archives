Seu, Mindy. “The Poetry of Tools”

Are.na blog post, 10 August, 2017.

With my frequent return to using the Web as a way to communicate the ways in which the reinvigoration of an archive and bringing these hidden-away cultural artifacts into a more active experience makes me constantly return to this blog post by Mindy Seu. She describes the ways in which the Internet has create platform-specific storytelling techniques that all create different sorts of engagements with data, interconnectivity, and the passage of time. The use of the tools of Web to create an viewing/interactive experience that is deeply tied to the networks of the Web are present in my Saving Grace and Human Artifacts projects.
